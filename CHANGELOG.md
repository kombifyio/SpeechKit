# Changelog

All notable changes to SpeechKit should be documented in this file.

The format is based on Keep a Changelog and this project is intended to ship under Apache-2.0.

## [Unreleased]

### Added

- host-managed Hugging Face token storage with user and install precedence
- OSS governance documents for licensing, contribution, security, and support
- explicit OSS release boundary documentation and export manifest

### Changed

- canonical Windows build now produces both the staged bundle and the NSIS installer
- release documentation now treats the framework core as provider-agnostic and tokenless

## [0.1.0] - TBD

First public framework-oriented release.

### Planned Scope

- Windows-first desktop host
- local SQLite-backed storage
- optional Hugging Face and VPS providers
- overlay-first runtime UX
- canonical Windows bundle and installer build

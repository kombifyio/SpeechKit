# Support

## Getting Help

Use the repository issue tracker for:

- reproducible bugs
- documentation gaps
- feature requests
- integration questions about the public framework surface

Before opening an issue:

- read [`README.md`](./README.md)
- read [`CONTRIBUTING.md`](./CONTRIBUTING.md)
- run the canonical build in [`scripts/build.ps1`](./scripts/build.ps1)

## What to Include

For technical support, include:

- Windows version
- SpeechKit version or commit
- whether you are using local, VPS, or Hugging Face providers
- exact reproduction steps
- logs or screenshots when relevant

## What This Repo Does Not Provide

- free hosted inference
- key management for third-party providers
- enterprise support guarantees
- support for private kombify-only integrations in the public issue tracker
